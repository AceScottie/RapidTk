21def example_basic_objects():
22 """
23 To create a rapidTk instance use the rapidTk() method which overwires the Tk() method and returns the Tk() method.
24 As it returns the Tk() method all commands on root are still valid if they are valid on a Tk() object.
25 rTk only uses Pack() methods over grid() and place(). grid() may be added in the future however place() will not be supported due to bad design useage.
26
27 PackProcess() is a class that lets you create all objecst in the correct order and the pack them in that order later.
28 to use, initilise the class and use the add() method to add widgets to the list and finally call the pack() method.
29 add() acceps all standard .pack() options and returns the widget unlike the original pack() method.
30 e.g. myFrame = pp.add(cFrame(root), side=TOP) #accepts rapidTk Widgets
31 myFrame2 = pp.add(Frame(root), side=TOP) #accepts Tkinter Widgets
32
33 c(Widget) is a class that overwites the original widget while keeping the widget accessible.
34 There may be addition options avaibleble in the cWidget however all original options are kept too keep it as compatible as possible.
35 All cWidgets, reWidgets and extended Widgets will have the methods of get_self(), get_parent() and get_root().
36 These methods will return the Widget, the widgets parent or the root Widget.
37
38 cLabel has 2 additional methods of get() and set() which get and set the text option.
39
40 cEntry() has an addition option of value which automatically sets up a StringVar textvariable and sets that to the value.
41 It also creates a popup rightclick menu for cut, copy, select all and paste.
42
43 """
44 root = rapidTk()
45 pp = PackProcess()
46 main = pp.add(cFrame(root), side=TOP, fill=BOTH, expand=1)
47 pp.add(cLabel(main, text="This is a basic rapidTk Label."), side=TOP, fill=X)
48 pp.add(cButton(main, text="This is an Example Button"), side=TOP)
49 pp.add(cEntry(main, value="Some Default Text"), side=TOP, fill=X)
50 pp.pack()
51 root.mainloop()

147def example_basic_menu():
148 """
149 cMenu is a quick and efficent way to make simple menus.
150 The only thing you need to provide is context on creation.
151 context is a dictionary of {Menu(.submenu(s)).Label : command}
152 example:
153 context={"Menu.Submenu1.Submenu2.options":lambda m="test":print(m)}
154 will create a main menu called "Menu" with a submenu called Submenu1 with a submenu called "Submenu2" with an option called "options"
155 The "options" option will run the command when clicked.
156
157 Note: subtree paths do not stack.
158 "Menu.Submenu1.Submenu2" is not the same as "Submenu1.Submenu2".
159 This will create 2 menus called "Menu" and "Submenu1"
160
161 all submenus and options can be accessed though the cMenu.sub_menus and cMenu.options dictionarys.
162 """
163 root = rapidTk()
164 root.geometry('320x150')
165 main = cFrame(root, side=TOP, fill=BOTH, expand=1)
166
167 menu_context = {
168 'File.Open': lambda m="Open":print(m),
169 'File.System.Advanced.Close': lambda m="Close":print(m),
170 'File.Quit': lambda m="Quit":print(m),
171 'System.Kill': lambda m="Kill":print(m),
172 'Exit': lambda m="Exit":print(m)
173 }
174 myMenu = cMenu(root, context=menu_context)
175 log = logging.getLogger('rapidTk')
176 log.rtklog(myMenu.sub_menus)
177 log.rtklog(myMenu.options)
178
179 root.config(menu=myMenu)
180 root.bind("<Button-3>", myMenu._do_popup) #bind to right click when widget clicked.
181 root.mainloop()
